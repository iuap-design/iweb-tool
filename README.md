# iweb

> a open-source tool for building resource.

## install

```
$ npm install iweb -g
```

## development

get help
```
$ iweb --help
```

init project
```
$ iweb init
```

build your project
```
$ iweb build
```

create a server for develop
```
$ iweb server -p 8080
```

publish your public resource
```
$ iweb publish
```
